java是一种面向对象、静态类型、编译执行，有VM/GC和运行时、跨平台的高级语言

## java、C++、Rust的区别
### C++

特点：自行管理内存、可以编写很自由的代码

缺点：一不小心内存泄露，导致程序崩溃

### java/Go

特点：内存生命周期由JVM运行时统一管理，绝大部分场景下，可以自由编写代码，不用关心内存情况

### Rust

特点：必须清楚明白的用Rust的规则管理好变量，好让机器能明白高效的分析和管理内存

缺点：代码不易理解，不自由，学习成本高
